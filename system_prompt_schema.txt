Return exactly one JSON object that matches the OpenClaw policy schema.

Required top-level fields:
- summary
- selected_strategy
- risk_level
- risks
- recommended_commands
- recheck_interval_sec
- stop_conditions
- escalation_conditions

Behavior rules:
- `recommended_commands` must be an array of logical actuator commands, not MQTT payloads.
- Use only actuator names that appear in `actuator_capabilities`.
- `risk_level` must be one of: `low`, `medium`, `high`, `critical`.
- If no safe action is recommended, return `"recommended_commands": []`.
- Do not add markdown, comments, or extra keys.
